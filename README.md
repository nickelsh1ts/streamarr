**Streamarr** is an open source web application meant to wrap an existing \*Arr ecosystem such as **[Sonarr](https://sonarr.tv/)**, **[Radarr](https://radarr.video/)**, **[Overseerr](https://overseerr.dev/)**, and **[Plex](https://www.plex.tv/)** enabling a smooth user friendly experience.

## Overview

Streamarr provides a unified dashboard for managing your Plex server alongside your \*Arr ecosystem. It simplifies common tasks like inviting users to Plex, tracking upcoming releases, monitoring downloads, and proxying access to your backend services‚Äîall from a single, polished interface designed for both administrators and end users.

## Major Features

- **Full Plex Integration** ‚Äî Sign in with Plex, embedded Plex Web player, user management
- **Robust Invite System** ‚Äî Usage limits, expiry dates, library access controls, QR codes
- **Release Calendar** ‚Äî Upcoming releases from Sonarr and Radarr, plus custom events
- **\*Arr Service Integration** ‚Äî Connect Radarr, Sonarr, Lidarr, Prowlarr, Bazarr, Tdarr, and more
- **Download Client Management** ‚Äî Monitor and manage qBittorrent, Deluge, and Transmission
- **Flexible Notifications** ‚Äî Email (with PGP encryption), Web Push, and In-App notifications
- **Internal Service Proxy** ‚Äî Securely access embedded \*Arr services through Streamarr
- **Granular Permissions** ‚Äî 15+ permission types for fine-grained access control
- **Trial Periods** ‚Äî Optionally restrict new users during a configurable trial period
- **Custom Theming** ‚Äî Fully customize colors and logo to match your brand
- **PWA Support** ‚Äî Install as a Progressive Web App on mobile devices
- **Multi-language Support** ‚Äî Localized interface
- **Interactive Onboarding** ‚Äî Welcome modals and interactive tutorials for new users with customizable content

## Planned Features

- **Jellyfin Support** ‚Äî Dual-backend support for both Plex and Jellyfin users
- **Help Center Customization** ‚Äî Fully customizable help center content
- **Email Newsletters** ‚Äî Customizable email newsletters and communications
- **Additional Services** ‚Äî Huntarr, Cleanuparr, and Kometa integrations
